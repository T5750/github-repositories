### Javaee-tutorial
[Javaee-tutorial](https://github.com/T5750/github-repositories/tree/master/Javaee-tutorial)是个简单的教务系统网站,并且结合了图书订购功能
- IoC容器: Spring 4.2.4.RELEASE
- web框架: Spring MVC 4.2.4.RELEASE
- orm框架: MyBatis 3.3.1
- 安全框架: Shiro 1.2.4
- 数据源: DBCP 1.4
- 日志: SLF4J 1.7.14
- Json: Gson 2.5
- 前端框架: Bootstrap v3.3.6
