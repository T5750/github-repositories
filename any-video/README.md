## any-video

Spring Boot 结合网络爬虫开发的完整视频网站，演示地址：http://www.ictgu.cn

### 开发环境
-------
- JDK 1.8
- MariaDB 10.1.22（MySql通用）
- Redis 3.2.8
- Maven 3.3.9
- 开发工具 intellij idea 2016
- 版本控制 Git
- 开发测试环境 Windows 10
- 生产运行环境 CentOS 7.2

### 技术框架
-------
- 核心：Spring Boot 1.5.1
- 数据：Mybatis
- 数据库连接池：Hikari
- 安全: Spring Serucity
- 模板：Thymeleaf 3
- 前端：Bootstrap  + Jquery
- 爬虫：Jsoup
- 容器: Undertow
